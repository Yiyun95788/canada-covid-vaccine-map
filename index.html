Canada COVID-19 Dashboard

This interactive dashboard visualizes COVID-19 mortality rates and vaccination data across Canadian provinces from 2020 to 2024.

How to Use This Dashboard:

  1. Click on any province to view detailed mortality and vaccination charts
  2. Use the year selector at the bottom to explore different time periods
  3. Toggle between mortality and vaccination heat maps using the STATISTICS dropdown
  4. Click the RETURN button to reset the map view
  5. Use the VISUALIZATIONS button to explore city-level data

Data sources: Public Health Agency of Canada, Statistics Canada
